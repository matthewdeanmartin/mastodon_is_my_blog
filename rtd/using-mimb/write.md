# Write

The **Write** page is the simplest part of Mastodon is My Blog (MIMB): it is a clean box for publishing a post.

## What it does

You type into the editor and press **Publish**.

After publishing:

- the post is sent to Mastodon
- MIMB refreshes its cache so the new post can show up locally

## What this page is good for

- short updates
- quick publishing without opening your instance's main interface
- staying in the same local app while reading and writing

## What to expect today

Write is intentionally lightweight right now.

That means:

- it is fast to understand
- it is not yet a full-featured long-form writing studio

The roadmap already calls for a better editor, so this area should become richer over time.
