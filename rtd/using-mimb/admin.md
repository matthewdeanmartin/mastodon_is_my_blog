# Admin

The **Admin** page is where Mastodon is My Blog (MIMB) becomes manageable instead of magical.

If People and Content are for reading, Admin is for **control**.

## Connection status

If your account is not connected yet, Admin is where you finish the OAuth login flow.

## Cache sync

**Force Refresh Cache** updates the local view of:

- follows and followers
- recent timeline activity
- recent notifications
- your own recent posts

Use this when the app feels out of date.

## Own account archive

**Initialize/Catch Up Own Account** goes deeper into your own posting history so your archive is more complete.

This is especially important if you want a stronger local record of your posts or plan to build the static blog export.

## Catch-up jobs

Catch-up is for broader backfilling.

### Catch Up Now (Urgent)

This is the faster, more aggressive pass.

### Start Trickle

This is the slower, gentler pass for deeper history.

### Stop

Stops the running catch-up job.

### Queue preview

The queue preview shows which accounts are likely to be processed first.

## Content Hub bundles

This section lets you build topic collections.

### New bundle

Create a named topic set.

### Terms

Each term is either:

- a **hashtag**
- a **search**

### Sync Server Follows

Imports hashtags you already follow on Mastodon as read-only groups.

### Edit and Delete

You can edit or remove your own bundles. Read-only server-follow groups cannot be deleted here.

## Best habit

If something in MIMB looks missing, stale, or oddly narrow, Admin is usually the first place to check.
