# Roadmap

Mastodon is My Blog (MIMB) is already usable, but it is still clearly growing.

## Current direction

The public roadmap currently includes:

- a much better editor
- improved blog roll filters
- improved post filters
- better account syncing strategies

## What that means for users

You should expect the core idea to stay the same:

- calm reading
- people-first browsing
- better topic grouping
- local control

But some of the details around editing, discovery, and syncing are still being sharpened.

## Areas likely to improve most

### Writing

The current Write page is intentionally simple. A better editor should make long-form posting and thoughtful drafting feel more at home.

### Reading filters

The app already has strong opinions about reading by person and post shape. The roadmap suggests those opinions will become more accurate and easier to trust.

### Sync behavior

Expect the balance between freshness, catch-up depth, and multi-account handling to improve over time.

## Best way to approach MIMB today

Use MIMB for the things it is already good at:

- reading by person
- finding topic clusters
- keeping a local archive mindset

Then treat newer edges as part of an active product, not a finished platform.
